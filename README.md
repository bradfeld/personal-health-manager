## Personal Health Manager

For years, I've been frustrated that all of my personal health information is not integrated into one app. While I fantasize that someone will get it right, no one has.

## Starting App

My running coach and I have developed a spreadsheet that I use which integrates summary level data from Training Peaks, Whoop, and inBody.

The first iteration of this will duplicate the functionality of that spreadsheet, but turn it into an actual system that automatically pulls the data from each system and displays it in a useful format.
